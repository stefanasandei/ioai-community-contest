# PyTorch & Deep Learning Frameworks

PyTorch is the most common framework in AI competitions and research. Its dynamic computation graph makes debugging intuitive and experimentation fast.

## Core concepts to master

- **Tensors:** the fundamental data structure in PyTorch.
- **Autograd:** automatic differentiation that powers backpropagation.
- **`nn.Module`:** how to define and compose neural network layers.
- **DataLoaders:** efficient batching, shuffling, and augmentation pipelines.
- **Training loops:** writing clean, reusable training and evaluation code.

## Tips for IOAI preparation

Most IOAI tasks can be solved with a small number of well-known architectures. Don't over-engineer; focus on understanding how to adapt standard models to new datasets and loss functions.

> Start with the official PyTorch tutorials, then use *Dive into Deep Learning* as a reference while you build projects.
