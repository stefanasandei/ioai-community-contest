# Getting Started with AI

If you're new to artificial intelligence, the sheer volume of topics can feel overwhelming. The key is to build a solid foundation before diving into competition-specific material.

## What to focus on first

1. **Intuition over memorization.** Understand *why* models work, not just *how* to call them.
2. **Code early, code often.** Reading about gradient descent is helpful; implementing it is transformative.
3. **Work with real data.** Use datasets from Kaggle, UCI, or Hugging Face to build intuition.

## Suggested path

Start with the fundamentals of machine learning, then pick up Python tooling (`numpy`, `pandas`, `matplotlib`), and move into deep learning once you're comfortable with the basics.
