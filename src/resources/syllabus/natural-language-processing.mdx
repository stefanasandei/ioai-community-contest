# Natural Language Processing

Language is one of the most important modalities in modern AI. This section covers both classical text processing and modern transformer-based approaches.

## Topics

- **Text Preprocessing:** tokenization, normalization, and preparing text for models.
- **Word Embeddings:** dense vector representations of words.
- **Language Models:** from n-grams to large pre-trained transformers.
- **Prompting & Inference:** how to effectively use and evaluate LLMs.

## Modern focus

Most contemporary NLP tasks are solved by fine-tuning or prompting pre-trained language models. Make sure you are comfortable with the Hugging Face ecosystem.
