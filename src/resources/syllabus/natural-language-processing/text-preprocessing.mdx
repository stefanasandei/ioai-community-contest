# Text Preprocessing

Raw text is messy. Preprocessing turns it into a clean representation that models can consume.

## Common steps

- **Tokenization:** splitting text into words, subwords, or characters.
- **Lowercasing and normalization.**
- **Removing or preserving punctuation and stopwords.**
- **Stemming and lemmatization.**

## Modern tokenization

Subword tokenization (BPE, WordPiece, SentencePiece) balances vocabulary size and coverage. Most transformer models use these methods.

## Tools

Practice with `nltk`, `spaCy`, and the Hugging Face `tokenizers` library.
