# Language Models

A language model predicts the probability of text. Modern language models are pre-trained on massive corpora and then fine-tuned for specific tasks.

## Types

- **Autoregressive models:** GPT-style, predict next token.
- **Masked language models:** BERT-style, predict masked tokens.
- **Encoder-decoder models:** T5, BART for translation and summarization.

## Fine-tuning

Most IOAI-style NLP tasks are best approached by fine-tuning a pre-trained model. Learn the Hugging Face `Trainer` API and how to adapt models to classification, NER, and generation tasks.
