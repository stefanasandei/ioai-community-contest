# Prompting & Inference

Large language models can be used without fine-tuning by crafting good prompts. This is increasingly relevant for rapid prototyping and certain IOAI-style tasks.

## Prompting techniques

- **Zero-shot and few-shot prompting**
- **Chain-of-thought prompting**
- **Role prompting and structured outputs**

## Inference considerations

- **Temperature, top-p, and top-k sampling**
- **Context window limits**
- **Latency and cost tradeoffs**

## Evaluation

Always evaluate prompted models with the same rigor as trained models. Prompting is not a replacement for validation.
