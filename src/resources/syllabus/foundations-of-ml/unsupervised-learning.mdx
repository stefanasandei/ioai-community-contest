# Unsupervised Learning

Unsupervised learning finds patterns in data without labeled targets. It is essential for exploration, preprocessing, and understanding structure.

## Main techniques

- **Clustering:** k-means, hierarchical clustering, DBSCAN.
- **Dimensionality reduction:** PCA, t-SNE, UMAP.
- **Anomaly detection:** identifying rare or unusual points.

## When to use it

Use clustering for customer segmentation, anomaly detection for fraud, and dimensionality reduction for visualization or preprocessing high-dimensional inputs.

## Practice ideas

- Cluster a dataset and interpret the resulting groups.
- Use PCA to visualize high-dimensional data.
- Build a simple anomaly detector and evaluate it.
