# Supervised Learning

Supervised learning is the task of learning a function that maps inputs to outputs based on example input-output pairs.

## Core algorithms

- **Linear regression** for continuous targets.
- **Logistic regression** for binary classification.
- **Decision trees, random forests, and gradient boosting** for structured data.
- **k-NN and SVMs** for smaller, well-defined problems.

## What to practice

1. Implement gradient descent for linear regression from scratch.
2. Train a classifier on a tabular dataset using scikit-learn.
3. Compare multiple algorithms using cross-validation.

## Key concepts

- Loss functions and optimization
- Overfitting vs. underfitting
- Bias-variance tradeoff
