# Feature Engineering Basics

Feature engineering is the process of transforming raw data into features that better represent the underlying problem to a model.

## Common transformations

- **Scaling and normalization**
- **Encoding categorical variables:** one-hot, target encoding, embeddings
- **Datetime features:** extracting day, month, cyclical encodings
- **Text features:** TF-IDF, count vectors
- **Interaction features:** combining existing features

## Mindset

Good features often matter more than complex models. Spend time exploring the data and thinking about what information would help a model make better predictions.

> Every column is a hypothesis about what the model needs to know.
