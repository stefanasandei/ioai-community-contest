# Neural Network Basics

A neural network is a composition of simple differentiable units. Understanding these basics makes every advanced architecture approachable.

## Building blocks

- **Neurons:** weighted sum of inputs passed through an activation function.
- **Activations:** ReLU, sigmoid, tanh, softmax.
- **Loss functions:** MSE, cross-entropy.
- **Backpropagation:** computing gradients efficiently using the chain rule.

## Exercises

1. Implement a single-layer perceptron in NumPy.
2. Build a small MLP in PyTorch and train it on MNIST.
3. Visualize how activations and gradients change during training.

## Watch out for

- Vanishing and exploding gradients
- Poor weight initialization
- Learning rates that are too high or too low
