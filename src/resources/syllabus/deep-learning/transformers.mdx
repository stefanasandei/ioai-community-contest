# Transformers

Transformers have become the foundation of modern NLP, vision, and multimodal AI. The key idea is self-attention, which lets the model weigh the importance of different input tokens.

## Key components

- **Self-attention:** relating different positions in a sequence.
- **Multi-head attention:** multiple attention mechanisms in parallel.
- **Positional encoding:** injecting order information.
- **Feed-forward layers and layer normalization.**

## Practical usage

Most practitioners use pre-trained transformers via Hugging Face. Learn how to fine-tune them and how to interpret attention patterns.
