# Computer Vision

Computer vision enables machines to interpret images and videos. This section covers the full spectrum from image classification to modern vision transformers.

## Subsections

- **Image Classification:** assigning a single label to an image.
- **Object Detection:** finding and labeling multiple objects.
- **Image Segmentation:** pixel-level understanding of images.
- **Vision Transformers:** applying transformer architectures to images.

## Practical tip

Start with transfer learning using pre-trained CNNs. They provide strong baselines with minimal data and compute.
