# Multimodal Models

Multimodal models process and generate information across multiple modalities, such as text, images, and audio.

## Key ideas

- **Joint embeddings:** mapping different modalities into a shared space.
- **Contrastive learning:** CLIP aligns images and text.
- **Multimodal transformers:** process mixed inputs in a single architecture.

## Applications

- Image captioning
- Visual question answering
- Text-to-image generation
- Speech-to-text with context

## Study approach

Understand CLIP first. It is conceptually simple and unlocks many downstream tasks.
